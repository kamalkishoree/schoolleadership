# Mobile App Assets

This directory should contain the following image assets:

## Required Assets

1. **icon.png** (1024x1024px)
   - App icon for iOS and Android
   - Square icon with transparent background

2. **splash.png** (1242x2436px recommended)
   - Splash screen image
   - Will be displayed on app launch

3. **adaptive-icon.png** (1024x1024px)
   - Android adaptive icon foreground
   - Should be centered with padding (safe area)

4. **favicon.png** (48x48px)
   - Web favicon

## Quick Setup

You can create placeholder images or use online tools to generate them:

1. Use a tool like [App Icon Generator](https://www.appicon.co/) to generate all assets
2. Or create simple placeholder images using any image editor
3. Place all files in the `assets/` directory

## For Development

If you don't have assets yet, you can temporarily comment out the asset references in `app.json` or create simple placeholder PNG files.
